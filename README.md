# elar-kstu

Проект ЭЛАР (Электронная Личная Академическая Регистр-система) — это современная цифровая платформа, созданная для КГТУ с целью полной цифровизации учебного процесса. Это не просто “ещё один электронный дневник”, а единая экосистема для всех участников образовательного процесса, объединяющая студентов, преподавателей, старост и администрацию в одном удобном, интуитивно понятном пространстве.
